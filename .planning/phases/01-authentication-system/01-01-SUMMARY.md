## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] Removed `vite-tsconfig-paths` from API Vitest config**
- **Found during:** Verification after Task 2
- **Issue:** The API package used CommonJS and `vite-tsconfig-paths` is ESM-only, causing Vitest config loading to fail.
- **Fix:** Removed the plugin and dependency because the scaffold does not use TS path aliases.
- **Files modified:** `apps/api/package.json`, `apps/api/vitest.config.ts`, `pnpm-lock.yaml`
- **Verification:** `pnpm test` and `pnpm --filter @lenglish/api test -- health` passed.
- **Committed in:** `96aab6d` and root lockfile commit context

**2. [Rule 3 - Blocking] Moved web health copy out of `page.tsx`**
- **Found during:** `pnpm build`
- **Issue:** Next.js App Router rejects arbitrary named exports from route files.
- **Fix:** Created `apps/web/src/app/home-health-copy.ts` and imported it from both the page and test.
- **Files modified:** `apps/web/src/app/page.tsx`, `apps/web/src/app/health.test.ts`, `apps/web/src/app/home-health-copy.ts`
- **Verification:** `pnpm build`, `pnpm test`, and web e2e passed.
- **Committed in:** `71e5bdf`

**3. [Rule 3 - Blocking] Ignored generated TypeScript build info**
- **Found during:** `pnpm typecheck`
- **Issue:** Web typecheck generated `apps/web/tsconfig.tsbuildinfo`, leaving the worktree dirty.
- **Fix:** Added `*.tsbuildinfo` to `.gitignore`.
- **Files modified:** `.gitignore`
- **Verification:** `git status --short` no longer reports the generated artifact.
