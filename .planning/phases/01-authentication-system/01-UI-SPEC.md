### Accent Reserved For
- Primary CTA buttons (Sign In, Sign Up, Save Profile)
- Active navigation / tab indicator
- Focus rings on inputs
- Selected avatar border
- Target TOEIC score card — selected state

### Gradient Usage
- Auth page background: subtle radial gradient `#7C3AED → #06B6D4` at 15% opacity over #0F172A
- Primary CTA button: linear gradient `#7C3AED → #6D28D9` (violet depth)
- Onboarding score cards: border gradient on hover

## Interaction Contracts

### 3D Flip Card (Login ↔ Signup)
```css
/* Card container */
.auth-card-container {
  perspective: 1200px;
}

/* Inner card that flips */
.auth-card-inner {
  transition: transform 600ms cubic-bezier(0.4, 0, 0.2, 1);
  transform-style: preserve-3d;
}

.auth-card-inner.flipped {
  transform: rotateY(180deg);
}

/* Front and back faces */
.auth-card-front,
.auth-card-back {
  backface-visibility: hidden;
  position: absolute;
  inset: 0;
}

.auth-card-back {
  transform: rotateY(180deg);
}
```

### Press-Down Button
```css
.btn-primary {
  transition: transform 120ms ease, box-shadow 120ms ease;
  box-shadow: 0 4px 0 #5B21B6; /* darker violet */
}

.btn-primary:hover {
  transform: scale(1.02) translateY(-1px);
}

.btn-primary:active {
  transform: scale(0.98) translateY(2px);
  box-shadow: 0 1px 0 #5B21B6;
}
```

### Pill Input
```css
.input-pill {
  border-radius: 9999px;
  padding: 12px 24px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.15);
  border: 1px solid transparent;
  transition: border-color 200ms, box-shadow 200ms;
}

.input-pill:focus {
  border-color: #7C3AED;
  box-shadow: 0 0 0 3px rgba(124, 58, 237, 0.2);
}
```

### Validation Shake
```css
@keyframes shake {
  0%, 100% { transform: translateX(0); }
  20%, 60% { transform: translateX(-6px); }
  40%, 80% { transform: translateX(6px); }
}

.input-error {
  animation: shake 400ms ease;
  border-color: #EF4444;
}
```

### Score Card Hover
```css
.score-card {
  transition: transform 200ms ease, box-shadow 200ms ease;
}

.score-card:hover {
  transform: translateY(-4px) scale(1.02);
  box-shadow: 0 8px 24px rgba(124, 58, 237, 0.3);
}

.score-card.selected {
  border: 2px solid #7C3AED;
  box-shadow: 0 0 16px rgba(124, 58, 237, 0.4);
}
```

- **Primary language:** Vietnamese — all UI labels, buttons, navigation

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Focus visible | 3px violet ring (#7C3AED at 20% opacity) on all focusable elements |
| Color contrast | Text primary #F8FAFC on #0F172A = 15.4:1 ratio ✓ |
| Password toggle | Eye icon button with aria-label "Show password" / "Hide password" |
| Form validation | aria-invalid, aria-describedby linking input to error message |
| Reduced motion | @media (prefers-reduced-motion: reduce) — disable flip, shake, floating animations |
| Screen reader | Form headings (h1), fieldset/legend for form groups |
