<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser UI to backend API | Untrusted form values enter API through fetch calls. |
| Middleware to protected routes | Navigation decisions happen before app route rendering. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-13 | Information Disclosure | Frontend auth code | mitigate | Never read or store JWT values in localStorage/sessionStorage. |
| T-1-14 | Spoofing | Route middleware | mitigate | Treat middleware as UX only; backend guards remain source of truth. |
| T-1-15 | Tampering | Reset form | mitigate | Send reset token only to backend confirm endpoint; do not persist it in client state beyond form submit. |
</threat_model>
